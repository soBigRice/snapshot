# 2D 随机靶子射击游戏设计文档

## 一、游戏概述

**游戏类型：** 2D 街机反应射击游戏

**核心玩法：**
玩家控制一个固定位置的炮台，炮口方向始终跟随鼠标移动。点击鼠标发射子弹，击中随机出现并随机消失的靶子获取分数。游戏核心是
**反应速度 + 精准瞄准 + 连击节奏**。

**核心乐趣：** - 快速反应 - 精准瞄准 - 目标优先级判断 - 连续命中的节奏感

------------------------------------------------------------------------

# 二、核心玩法机制

## 1. 固定发射点

发射器固定在屏幕底部中央。

特点： - 玩家无需移动角色 - 只需控制瞄准方向 - 玩法简单但节奏快

示意：

          靶子区域

       ○     ○       ○

              ○

    ----------------------
            炮台

------------------------------------------------------------------------

## 2. 鼠标控制方向

炮口始终指向鼠标位置。

计算方式：

    direction = normalize(mousePosition - turretPosition)

点击鼠标左键即可发射子弹。

------------------------------------------------------------------------

## 3. 子弹机制

子弹规则：

-   直线飞行
-   固定速度
-   可设置射击间隔（例如 0.12 秒）
-   命中靶子后消失

------------------------------------------------------------------------

# 三、靶子系统

靶子会随机生成，并在一定时间后消失。

每个靶子包含：

-   随机位置
-   随机存在时间
-   不同类型
-   不同得分

------------------------------------------------------------------------

## 1. 普通靶

特点： - 中等大小 - 中等存在时间 - 常规目标

分数：

    +100

------------------------------------------------------------------------

## 2. 快闪靶

特点：

-   存在时间很短
-   出现突然
-   分数高

存在时间：

    0.6s ~ 1.2s

分数：

    +250

------------------------------------------------------------------------

## 3. 大型靶

特点：

-   体型大
-   容易命中
-   分数较低

分数：

    +50

作用：

-   稳定游戏节奏

------------------------------------------------------------------------

## 4. 小型高分靶

特点：

-   体型小
-   难命中
-   分数高

分数：

    +300 ~ +400

------------------------------------------------------------------------

## 5. 陷阱靶

特点：

-   不应该射击
-   击中会扣分

分数：

    -200

作用：

增加识别压力。

------------------------------------------------------------------------

# 四、靶子生成系统

靶子随机生成规则：

生成时间：

    0.4s ~ 1.0s

生成区域：

-   屏幕上半部分
-   避免靠近炮台
-   避免重叠

最大数量：

    同屏最多 5 个靶子

------------------------------------------------------------------------

# 五、靶子生命周期

每个靶子都有生命周期：

    生成
     ↓
    存在 1~3 秒
     ↓
    若未被击中则自动消失

当靶子快消失时：

-   边框闪烁
-   透明度降低
-   提示玩家优先攻击

------------------------------------------------------------------------

# 六、计分系统

基础分：

  靶子类型   分数
  ---------- ------
  普通靶     +100
  大型靶     +50
  快闪靶     +250
  小型靶     +300
  陷阱靶     -200

------------------------------------------------------------------------

# 七、连击系统

连续命中会提高倍率。

示例：

    1~4 连击    x1
    5~9 连击    x1.5
    10+ 连击    x2

若发生以下情况：

-   靶子消失
-   射空
-   打到陷阱

则 **Combo 清零**。

------------------------------------------------------------------------

# 八、游戏模式

## 限时模式

游戏时长：

    60 秒

规则：

-   时间结束结算分数
-   挑战更高分

------------------------------------------------------------------------

## 生存模式

规则：

-   漏掉靶子会扣生命
-   生命归零游戏结束

------------------------------------------------------------------------

## 连击挑战

规则：

-   只有连击才能得高分
-   一旦漏掉目标倍率清零

------------------------------------------------------------------------

# 九、视觉反馈

命中反馈：

-   靶子缩放
-   粒子爆炸
-   分数数字弹出

射击反馈：

-   枪口火焰

-   炮台轻微后坐

Combo反馈：

-   屏幕显示连击数
-   连击越高字体越夸张
